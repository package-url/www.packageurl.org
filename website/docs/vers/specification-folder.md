# Specification

The VERS specification will be submitted to Ecma International as a new
standard in 2026.

The key elements of the current VERS specification are:
- **Core Specification**: This is the human-friendly and normative
specification for VErsion Range Specifier.
- **How to parse and validate VERS**: Instructions for validating and parsing
a VERS.
- **Version schemes**: Summary of **version-schemes** for many package
ecosystems and vulnerability databases.
- [VERS Test JSON Schema](/docs/vers/schemas.md)
